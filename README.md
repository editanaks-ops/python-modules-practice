# Python Modules Practice

Учебный репозиторий для练ировки работы с модулями, пакетами и `__init__.py`.

## Содержимое
- **library_manager** — домашнее задание: пакет для управления библиотекой  
  - добавление и удаление книг  
  - поиск по различным критериям  
  - генерация отчёта  
- **practice_1**, **practice_2** — практические задания по модулям и пакетам  
- **test_library.py** — пример использования пакета `library_manager`

## Основные компоненты
- `Library` — класс для управления книгами  
- `generate_report` — формирование текстового отчёта  
- `utils` — валидация и форматирование данных  

Репозиторий создан для отработки структуры Python-проектов и импортов.


Purpose of This Repository

Practice creating and organizing Python modules

Learn package structure and imports

Understand __init__.py and how to control exports

Build a complete, working custom Python package
